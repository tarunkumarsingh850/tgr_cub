SMTP by User or Company

This module will allow to configure outgoing mail server by user or by multiple companies.

- To specify outgoing mail server by user or company, need to configure the boolean field first.
- Can get the configuration inside Settings >> general settings >> SMTP configuration


Let's configure outgoing mail server with company

- Can specify out going main server with , multiple companies

- For that need to select the SMTP By Company First from settings

- After selecting the SMTP By Company configuration to true

- Company field will be visible in Outgoing mail server

- Now can configure outgoing mail server with multiple companies

Let's configure outgoing mail server with user

- Can specify out going main server with Specific User

- For that need to select the SMTP By User First from settings

- After selecting the SMTP By User configuration to true

- User field will be visible in Outgoing mail server

- Now can configure outgoing mail server with specific user
